INTRODUCCIÓN

Esta pagina tiene como fin dar a conocer el proceso de desarrollo de sitios web en formato HTML5 y CSS3,
para ser visualizados en diferentes navegadores teniendo en cuenta su funcionabilidad y usabilidad,
por medio del uso de diferentes editores de código propuestos en el desarrollo del curso.
Esto con el fin de fortalecer los conocimientos en herramientas de diseño web y herramientas a la vanguardia de la tecnología.

OBJETIVOS DEL OVI

1.Desarrollo de sitios web en formato HTML5 y CSS3.
2.Fortalecer el manejo de Sistemas de control de versiones y Editores de código para la web.
3.Realizar la Configuración de forma correcta el Contador de visitas para el sitio web del OVI.
